# Tickets Documentation Notes

This folder is the **public extraction workspace** for AgentLoops documentation.

Only include entries that are useful to other developers integrating or contributing
to AgentLoops:

- extraction plans and readiness checklists,
- product/positioning docs,
- public architecture notes,
- implementation notes that describe reusable concepts and contracts,
- examples of workflow and validation strategy.

Please avoid project-specific ticket records, private evidence, internal
operations notes, and raw Inti Reader incident write-ups. If a document is
Inti-only context, keep it in `inti-docs` instead of this repo.

If you need to preserve internal history, create a clearly private path in the
host repo rather than adding it under `docs/tickets`.
